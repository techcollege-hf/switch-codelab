Små hurtige SWITCH opgaver.

CSS skal skrives i filen: /opgavefiler/js/easy.css

Opgave 1

Instruktioner

  1. skriv en switch struktur der kan vurdere hvilket tal brugeren har tastet ind.
  2. skriv følgende ud som info, i konsollen: "du har valgt: (det valgte tal) "
  3. skriv evt. en view funktion der kan vise teksten i elementet "myAnsverFeedbackElement".

opgave svar her.



    switch(expression) {
        case x:
            // code block
            break;
        case y:
            // code block
            break;
        }  

        // view function
        function showMessage(myMessage){
            myAnsverFeedbackElement.innerHTML=`<p>${myMessage}</p>`;
        }

Opgave 2

Instruktioner

  1. Tilføj en default case til koden i opgave 1, der viser hvilket talbrugeren har tastet.
    Og en besked om at tallet ikke er tilladt. 


    switch(expression) {
        case x:
            // code block
            break;
        case y:
            // code block
            break;

        default:
            // code block
        }

            myAnsverFeedbackElement.innerHTML=`<p>${myMessage}</p>`;
        }
